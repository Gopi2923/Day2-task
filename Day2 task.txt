List 5 difference between Browser JS(console) v Nodejs

1. NodeJS :
NodeJS is a cross-platform and opensource Javascript runtime environment that allows the javascript to be run on the server-side. Nodejs allows Javascript code to run outside the browser. 
Nodejs comes with a lot of modules and mostly used in web development.

2. JavaScript :
Javascript is a Scripting language. It is mostly abbreviated as JS. It can be said that Javascript is the updated version of the ECMA script. 
Javascript is a high-level programming language that uses the concept of Oops but it is based on prototype inheritance.
